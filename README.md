## **Project setup guideline**
### Frontend
Step 1: Clone the Repository
``` git
https://github.com/Mosharof-Hossen/medi-travel.git
```

Step 2: Install Dependencies
``` javascript
cd client
npm install
```
Step 3: Run
```javascript
npm run dev
```

### Backend
Step 1: Clone the Repository
``` git
https://github.com/Mosharof-Hossen/medi-travel.git
```

Step 2: Install Dependencies
``` javascript
cd server

```

Step 3: Set Up Backend Environment Variables
